# IEDC SB College Website

A modern, minimal website for the Innovation and Entrepreneurship Development Cell (IEDC) of Saint Berchmans College.

---

## 🚀 Tech Stack

- Next.js (App Router)
- Tailwind CSS
- Framer Motion

---

## ✨ Features

- Clean, minimal UI inspired by YC / Stripe
- Fully responsive (mobile-first design)
- Smooth animations and micro-interactions
- Structured "Get Involved" system with dedicated pages
- Contact page with FAQ and section navigation
- Reusable component-based architecture
